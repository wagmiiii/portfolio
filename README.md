# Ademola Ajala | Personal Portfolio

Welcome to the repository for my personal portfolio! This project is a highly dynamic, cinematic, and fully responsive website designed to showcase my work as an AI Native Software Engineer. 

You can view the live site here: [wagmiiii.github.io/portfolio](https://wagmiiii.github.io/portfolio)

## 🚀 Tech Stack

This portfolio was architected entirely from scratch using native web technologies, ensuring maximum performance, zero bloat, and complete control over the layout and animations.

*   **HTML5:** Semantic structuring for flawless accessibility and SEO.
*   **CSS3:** Native CSS variables for real-time Light/Dark mode toggling, dynamic fluid typography (`vw` scaling), flexbox/grid layouts, and cinematic scroll animations.
*   **Vanilla JavaScript (ES6+):** Utilizes `IntersectionObserver` for highly performant scroll-triggered entrance animations, and `localStorage` with fail-safes to persistently track user theme preferences. No heavy frameworks (React/Vue) were used.

## 🎨 Features

*   **Cinematic Video Backgrounds:** High-resolution MP4 backgrounds with advanced CSS masking and blend modes.
*   **True Dark/Light Mode:** Instantaneous theme toggling driven entirely by CSS root variables.
*   **Responsive Brutalism Typography:** Massive typographic elements that dynamically scale down to fit even the smallest mobile screens without overflowing.
*   **Bulletproof Fail-safes:** Advanced JavaScript error handling ensures that strict browser privacy modes (which block cookies or `localStorage`) will never crash the site's critical functionality (like the mobile burger menu).

## 🌐 Deployment (GitHub Pages)

This project is deployed using **GitHub Pages**, taking advantage of the fact that it is a purely static site.

If you are cloning this repository and want to host it yourself:
1. Commit your changes and push the repository to GitHub on the `main` branch.
2. Navigate to your repository on GitHub.
3. Click the **Settings** (gear) icon in the top right.
4. On the left sidebar, scroll down to **Pages**.
5. Under the **"Build and deployment"** section, set the Source to **"Deploy from a branch"**.
6. Under the **Branch** dropdown, select `main` and the `/ (root)` folder.
7. Click **Save**. 

GitHub Actions will automatically build and deploy your site within 60 seconds!
